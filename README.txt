Inorder to start a server and run the application

Do the following:

1) If it's the first time you are going to run the application then run
"bash setup.sh"

2)If you have already done that then you can start the server by running the run.sh file

3) If you want to shutdown the server , run the command displayed while starting the server to shutdown the server
---> Follow the instruction that says something like this-> 'kill -SIGTERM 2131' and run it on the terminal.
